# Roadmap V2 - Features Comerciales

**Fecha**: 2025-01-30
**Objetivo**: Transformar Mentor de Proyectos en una app indispensable y comercialmente exitosa
**Visión**: El "segundo cerebro" para ideas + el "coach" para ejecución

---

## Resumen Ejecutivo

### Propuesta de Valor Actual
- IA planifica proyectos
- Check-ins diarios
- Revisiones semanales

### Propuesta de Valor V2
- **Captura**: Todas tus ideas en un lugar (Idea Backlog)
- **Planifica**: IA divide en pasos cuando estés listo
- **Ejecuta**: Check-ins, focus timer, accountability
- **Reflexiona**: Analytics, patrones, crecimiento
- **Celebra**: Logros compartibles, gamificación

## Catálogo Completo de Features

### Categoría 1: IDEA BACKLOG (Captura de Ideas)

| ID | Feature | Descripción | Prioridad |
|----|---------|-------------|-----------|
| IB-01 | **Quick Capture** | Agregar idea en 5 segundos (título + nota opcional) | P0 |
| IB-02 | **Idea List View** | Vista de todas las ideas con búsqueda y filtros | P0 |
| IB-03 | **Adjuntos** | Subir PDFs, imágenes, screenshots, links a cada idea | P1 |

| IB-05 | **Chat IA por Idea** | Conversar con IA sobre una idea específica sin crear proyecto | P1 |
| IB-06 | **Idea Reminders** | Notificaciones periódicas: "¿Recuerdas esta idea?" | P2 |
| IB-07 | **Promote to Project** | Convertir idea en proyecto activo con un tap | P0 |

| IB-09 | **Idea Voting/Priority** | Ordenar ideas por prioridad personal | P2 |
| IB-10 | **Voice Note Ideas** | Capturar ideas por audio, transcripción automática | P3 |

**Límites de monetización**:
- FREE: 5 ideas máximo
- PRO: Ideas ilimitadas + adjuntos + IA chat
